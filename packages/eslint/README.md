ESLint config for Cube Dev Inc. repositories

## Installation

```bash
yarn add -D @cubedevinc/eslint-config
npx install-peerdeps @cubedevinc/eslint-config
```

## Usage

Create an `.eslintrc` in your project with the following:

```json
{
  "extends": "@cubedevinc"
}

```
